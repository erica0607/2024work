個人帳號

4篇文章

註冊會員帳號、登入及登出辦法

華冠投顧Clickgo 共有 4 種註冊會員帳號方式：

1.綁定 Facebook ID 快速註冊(尚未開放)
2.綁定 Google ID 快速註冊
3.使用LINE ID 快速註冊
4.使用 帳號和密碼 註冊

註冊流程

選擇「社群帳號綁定」或「電子信箱＋密碼」方式註冊，並完成信箱驗證。

若先前已經註冊會員帳號，請點擊「登入」輸入帳號及密碼進入您的會員帳號

綁定 Facebook 註冊，登入時請點選「 Facebook 」並請確認網頁、App已登入相同Facebook 帳號
綁定 Apple ID 註冊則點選「 Apple 」

登出會員帳號步驟

網頁版請點選右上角「個人頭像」>「登出」
